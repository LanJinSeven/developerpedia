# 获得开发者权限

钉钉[开发者后台](https://open.dingtalk.com)是应用开发最重要的控制台。通过开发者后台，钉钉开发者可以创建应用，进行应用开发。其中，应用的能力包括展示在工作台的网页应用、机器人、酷应用等。

当前开发者后台中，应用开发的功能依赖**“应用开发子管理员”**权限，本文介绍如何获得该权限。如果没有该权限，进入应用开发时候就会出现相应的错误提示：
