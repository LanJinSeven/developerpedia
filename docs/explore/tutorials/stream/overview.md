# 概述

本文档介绍如何通过开发一个钉钉机器人，具备收发消息能力。

## 教程列表

通过学习本教程，你可以开发一个钉钉聊天机器人，具备以下能力：

* 接收聊天会话中的消息内容，包括单聊和群聊。备注：群聊仅限 AT 机器人的消息。
* 回复消息，或者主动发送消息到聊天会话中

|-------------------------------------------------------|---|
| [Go](/docs/explore/tutorials/stream/bot/golang/intro) | https://github.com/open-dingtalk/stream-tutorial-go |
